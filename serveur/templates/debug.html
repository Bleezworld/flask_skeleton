If you want to debug the frontend without caring about the backend, you can fill in the values you want here and the backend will take it into account.

Urls will be accessible on /api/debug/xxx.

All urls have to start with "/debug/". Ex: "/debug/aa", "/debug/aa/bb/cc".

Any JSON valid data is ok.

  • {"/debug/a": ["kjkj", "kjkj"], "/debug/b": ..., }
  • {"/debug/a": {"a": [1, 3, 4], "b": {"c": "d", "e": "f"}}, "/debug/b": ..., }
