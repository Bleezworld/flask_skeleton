Toutes les urls doivent commencer par "/debug/". Ex: "/debug/aa", "/debug/aa/bb/cc".

Les urls seront accessibles à /api/debug/xxx.

On peut mettre ce qu'on veut dans les paramètres tant que c'est correct en JSON : tableaux, dictionnaires, tableaux de dictionnaires, ...

  • {"/debug/a": ["kjkj", "kjkj"], "/debug/b": ..., }
  • {"/debug/a": {"a": [1, 3, 4], "b": {"c": "d", "e": "f"}}, "/debug/b": ..., }
